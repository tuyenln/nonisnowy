Fix image sizes after you have changed image sizes from Media Settings.

== Description ==

Once you changed image sizes in Media Settings, it allows you to fix all images to that size.

1. Login to admin panel,Go to Appearance => Acme Fix Images.
2. Check the image sizes you want to fixed
2. Click Fix All Images.

== Installation ==

**Method 1: Automatic Plugin Installation**

1. Login to admin panel,Go to Plugins => Add New.
2. Search for "Acme Fix Images" and install it.
3. Once you install it, activate it

**Method 2: Manual Plugin Installation**

1. Upload the plugin's folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. From WP Admin go to Appearance > Acme Fix Images.


== Frequently Asked Questions ==

= What you develop this plugin?  =

This plugin is specially developed for as support of our themes SuperMag and Acmeblog and other themes. But you can use this with any themes and any sites.

= Why you develop this plugin although there are other similar plugin work like this?  =

This plugin hard crop the default Image sizes: Thumbnail, Medium and Large, which other plugin doesn't do.
